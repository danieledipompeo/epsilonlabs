Module Descriptors

Swarm Intelligence (SWIN) 2014/5

Module Code ELE00068M
Lecturers TBA
Taken By ACS, ARE, MEng CSAI 4, MEng CSBES 4, MEng CSESE 4, MEng CSSE 4, MEng CSYS 4, MMath 4
Number of Credits 10
Terms Autumn
Teaching Aut 2-10
Closed Assessment [100%] Coursework (see information below), 30.00 hours

Workload

Assessment

Closed Assessment

  • Coursework (see information below), 30.00 hours

Aims

SWIN (ELE00068M) is an Electronics module from 2014+ and information about this module is available here:

http://www.elec.york.ac.uk/internal_web/gsp/are/modules/modules.html

Any issues in accessing the above pages, please contact Trudy in Electronics (email, trudy.hilary@york.ac.uk; telephone, 01904 322320), working days Tuesday, Wednesday and Thursday.

Note further there is a google site module page (containing relevant course material, academic staff contact info etc) also:
https://sites.google.com/a/york.ac.uk/swarm-intelligence/home

The aim of this module is to introduce the basic biology, algorithms, and uses of a range of swarm intelligence approaches. This is a growth area, and new approaches and advances appear regularly; the coverage of the module is flexible to allow new approaches to be added. Students will become familiar with the basic swarm intelligence literature, the underlying biological principles and their application in a number of engineering scenarios.

Assessment
This is 1 assessment by coursework (100%) - hand out in AuT/9; hand in SprT/3
(However please check Electronics pages for the most up-to-date information in terms of the module arrangements).
